# Cómo agregar tu solución a un ejercicio

Esta guía te muestra cómo subir tu archivo `.do` en menos de 5 minutos
usando solo el navegador. 

No hace falta saber Git para esto. Solo seguí los pasos y usá la interfaz web.

------------------------------------------------------------------------

## 🧭 Resumen rápido

Vas a hacer esto:

-  Crear una copia del repositorio (**Fork**)
-  Subir tu archivo a la web
-  Crear un **Pull Request (PR)** para que el profesor lo revise y acepte en el repositorio oficial

------------------------------------------------------------------------

## 1) Hacer un Fork

1.  Entrá al repositorio de la materia
2.  Hacé click en el botón **Fork** (arriba a la derecha)
3.  Se va a crear una copia en tu cuenta de GitHub

------------------------------------------------------------------------

## 2) Ir a la carpeta del ejercicio

Dentro de tu fork:

1.  Navegá a:

        ejercicios-optativos/ejercicio-xx/soluciones_alumnos/

2.  (Reemplazá `ejercicio-xx` por el ejercicio correspondiente)

------------------------------------------------------------------------

## 3) Subir tu archivo

1.  Click en **Add file** → **Upload files**

2.  Arrastrá tu archivo `.do`

3.  Nombre del archivo (OBLIGATORIO):

        apellido_nombre.do

    Ejemplo:

        perez_juan.do

4.  Abajo, en "Commit changes", dejá el mensaje por defecto

5.  Click en **Commit changes**

------------------------------------------------------------------------

## 4) Crear el Pull Request

1.  GitHub te va a sugerir crear un PR automáticamente → click en
    **Compare & pull request**

Si no aparece: - Andá a la pestaña **Pull requests** - Click en **New
pull request**

2.  Verificá que diga:
    -   base repository: repo de la materia
    -   head repository: tu fork
3.  Click en **Create pull request**

------------------------------------------------------------------------

## 5) Listo ✅

Tu entrega ya está enviada.

-   El profesor la va a revisar
-   Si está todo bien, se va a agregar al repositorio
-   Vas a poder ver las soluciones de tus compañeros

------------------------------------------------------------------------

## ⚠️ Reglas importantes

-   Un archivo por alumno
-   Usar el nombre correcto: `apellido_nombre.do`
-   Subirlo en la carpeta correcta
-   No modificar archivos de otros

------------------------------------------------------------------------

## ❓ Problemas comunes

**No veo el botón Fork** → Tenés que estar logueado en GitHub

**Subí el archivo en otra carpeta** → Podés borrarlo y volver a subirlo

**Me equivoqué en el nombre** → Podés reemplazar el archivo haciendo
otro commit
